# netsnmp-sys

> Low-level Rust FFI bindings to the Net-SNMP C library (autogenerated).

This crate provides autogenerated, unsafe bindings to the Net-SNMP C library so Rust code may call the native Net-SNMP APIs.

**Prerequisites**
- A system installation of Net-SNMP (development headers and libraries).
- `net-snmp-config` available on `PATH` (the build script invokes it to detect include/lib flags).
- Rust toolchain (`cargo`, `rustc`).

macOS example:

```bash
brew install net-snmp
net-snmp-config --configure-options
```

Debian/Ubuntu example:

```bash
sudo apt-get install libsnmp-dev
```

**Build**

From this crate root:

```bash
cargo build
```

`net-snmp-config` is used for CLFAGS and must be on the PATH.
